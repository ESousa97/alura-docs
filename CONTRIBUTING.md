# Contribuindo

Obrigado por contribuir!

## Fluxo
1. Abra uma issue descrevendo a mudança.
2. Crie um branch a partir de main.
3. Garanta que lint e testes passam:
   - npm run lint
   - npm test
4. Abra um Pull Request com descrição clara.

## Padrões
- Use Node.js 18+
- Não inclua segredos no código
